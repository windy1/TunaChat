Available commands:
  * /quit                   - Stops the client
  * /connect <host> [port]  - Connects to a chat server
  * /auth <user> <pass>     - Authenticates with the
                              server
  * /tell <user> <text>     - Sends a message
  * /list                   - Lists users currently
                              online
  * /disconnect             - Disconnects from the
                              current server
  * /clear                  - Clears the current output
  * /pref <key> <value>     - Sets a user preference
  * /prefs                  - Lists user preferences
  * /help                   - Lists available commands
